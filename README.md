## รายงานผลการทดสอบระบบ (Test Cases)
1. **Live Preview:** สามารถพิมพ์ชื่อแล้วระบบแสดงตัวอย่างผลลัพธ์แบบเรียลไทม์ได้ถูกต้อง
2. **Form Validation:** ระบบล็อกไม่ให้ส่งฟอร์มว่าง และแสดงข้อความเตือนชัดเจนใกล้ช่องกรอกข้อมูล
3. **Form Submission:** เมื่อกรอกข้อมูลถูกต้อง ระบบทำการเพิ่มรายการลงในกล่องรายการทั้งหมด และรีเซ็ตฟอร์มกลับสู่ค่าเริ่มต้นอย่างสมบูรณ์
